# Design System

## Theme

A compact, native-feeling developer tool for prolonged inspection on desktop monitors. Light and dark themes use restrained neutral surfaces with semantic colors reserved for roles and states.

## Typography

- System sans-serif for interface text.
- System monospace for identifiers, code, arguments, and structured values.
- Compact fixed sizing with clear weight changes rather than display typography.

## Color

- Neutral zinc surfaces and separators define the three-pane workspace.
- The primary accent marks selection and focus.
- User, assistant, thinking, tool, system, and metadata roles use semantic tokens.
- Icons and labels accompany semantic colors.

## Components

- Dense toolbar and segmented filters.
- Virtualized timeline and conversation lists.
- Expandable tool calls and structured JSON.
- Unframed detail sections separated by rules, not nested cards.
- Visible keyboard focus and standard interactive states.

## Layout

- Resizable three-pane desktop workspace.
- Timeline supports rapid filtering and scanning.
- Conversation shows turn boundaries and causal message order.
- Detail view prioritizes normalized execution information, with raw JSON always available.

## Motion

- Motion only communicates expansion, selection, and navigation.
- Use short ease-out transitions and respect reduced motion.
