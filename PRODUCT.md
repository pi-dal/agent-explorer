# Product

## Register

product

## Users

Developers and operators inspecting agent sessions to understand exactly what information an agent received, which actions it took, what each tool returned, and how the final response was produced.

## Product Purpose

Agent Explorer turns JSONL session logs into a navigable execution trace. Success means a user can reconstruct an agent's working process without reading raw JSON line by line.

## Brand Personality

Precise, restrained, technical.

## Anti-references

Avoid marketing-style dashboards, decorative data cards, playful visual metaphors, and interfaces that hide execution details behind summaries.

## Design Principles

1. Show causality before metadata.
2. Preserve raw evidence alongside normalized views.
3. Make dense execution traces easy to scan repeatedly.
4. Use consistent roles and states across supported log formats.
5. Keep the interface quiet so session activity remains the focus.

## Accessibility & Inclusion

Target WCAG AA contrast. Do not communicate roles or execution states by color alone. Respect reduced-motion preferences and maintain keyboard-accessible navigation.
